# 产品分析文档：智能学习管理助手

**文档版本**：V1.0  
**产品类型**：家庭教育SaaS工具  
**目标用户**：5-15岁孩子 + 家长  
**核心价值**：计划制定 → 打卡执行 → 积分激励 → 数据追踪 → 多端同步

---

## 一、产品定位

面向家庭场景的**学习与习惯养成管理工具**，通过游戏化激励（积分/成就/愿望清单）和数据可视化，帮助孩子提升自驱力，帮助家长轻松掌握成长进度。

### 差异化优势

- 家长审核机制（积分“待审定”）
- 正负激励并存（奖励+扣分）
- 艾宾浩斯记忆曲线复习计划
- 多账号分离与共享

---

## 二、用户角色与核心场景

| 角色 | 核心痛点 | 使用场景 |
| :--- | :--- | :--- |
| **孩子** | 拖延、缺乏目标感 | 查看今日任务 → 打卡/计时 → 获得积分 → 兑换愿望 |
| **家长** | 没时间监督、难以量化成长 | 创建计划 → 审核积分 → 查看统计图表 → 管理愿望 |

## 四、核心业务流程

### 4.1 学习计划打卡完整流程

1. 家长创建计划
2. 判断计划类型：
   - **计时任务**：孩子点击开始计时 → 正计时/倒计时/番茄钟 → 结束/暂停续时 → 自动记录时长
   - **无计时任务**：孩子点击快速完成
3. 判断是否需要审定：
   - **需要**：家长端收到待审定 → 家长确认通过 → 积分到账
   - **不需要**：积分直接到账
4. 更新连续打卡天数
5. 更新统计图表

### 4.2 积分流转路径

完成任务 → 获得基础星星 → （全勤额外星星） → 星星余额增加  
解锁成就 → 获得成就星星 ↗  
兑换愿望 → 星星余额减少 → 生成兑换记录 → 剩余次数减1

### 4.3 习惯养成正负激励逻辑

| 行为类型 | 示例 | 积分变化 | 审核方式 |
| :--- | :--- | :--- | :--- |
| 正向习惯 | 早睡、阅读、运动 | +1 ~ +5 | 孩子打卡即得 |
| 负向习惯 | 迟到、挑食、玩游戏超时 | -1 ~ -3 | 家长记录扣分 |

## 五、页面结构（信息架构）

### 首页（今日看板）

- 连续打卡天数 & 累计完成计划数
- 今日学习时间 / 户外时间
- 今日任务数量（已完成X/总数Y）
- 完成率进度条
- 快捷入口：查看详细介绍 / 图表统计 / 成就勋章 / 成绩追踪
- 计划列表（按周/日展示）
  - 任务名称 & 时间段
  - 操作按钮：开始计时 / 快速完成
  - 已完成任务显示：完成时间、累计时长、待审定角标

### 学习计划页

- 周视图（周一至周日）
- 我的学习计划列表
- 分享任务 / 批量添加
- AI创建 / 总预览
- 添加计划（选择分类、周期、时间段、积分）

### 图表统计页

- 最近7天 / 30天 Tab
- 总任务数 / 已完成数 / 平均完成率
- 总时长 / 日均时长
- 每日任务完成情况柱状图
- 最活跃日期标识

### 成绩追踪页

- 考试总数 / 平均得分率 / 优秀次数
- 趋势分析折线图
- 科目管理 / 添加成绩
- 考试成绩列表（搜索/筛选）
- 每场考试显示：得分/总分、排名、评语

### 积分与成就页

- 星星余额（本周/本月新增、已消费）
- 今日还可获得星星数 + 剩余任务明细
- 成就系统（已解锁勋章墙）
- 积分历史（收入/支出明细）
- 愿望清单（可兑换商品、剩余次数、兑换按钮）

### 习惯养成页

- 数据统计（累计打卡次数）
- 习惯管理（创建/编辑习惯）
- 习惯列表（搜索）
- 每个习惯：今日进度、打卡/记录按钮、奖惩积分显示
- 负向习惯：记录扣分入口

### 我的/设置页

- 多账号管理（添加孩子/切换角色）
- 多端同步状态
- 家长审定中心（待审积分列表）
- 全勤奖励规则设置
- 成就自定义配置

## 九、产品差异化亮点总结

1. **积分待审定机制**：家长可控制积分发放，保证任务完成质量
2. **正负激励并存**：既有奖励（早睡+2星），也有扣分（迟到-2星）
3. **艾宾浩斯记忆曲线**：针对知识点/语言的自动复习计划
4. **多端分离又共享**：孩子专注执行，家长管理审核
5. **图表可视化“孩子友好型”**：勋章、星星、进度条清晰易懂
